<h2 align='center'>Definición</h2>

Los tags en HTML (`etiquetas`) son marcadores especiales entre `< >` que estructuran y dan formato al contenido de una página web, indicando al navegador cómo mostrar elementos como títulos, párrafos, imágenes y enlaces, usando etiquetas de apertura (`<p>`) y cierre (`</p>`), aunque algunas se autocierran (`<br>`). Son el lenguaje fundamental para construir la estructura de cualquier sitio web.

<br></br>

<h2>¿Cómo funcionan?</h2>

* **Marcado:** Se colocan alrededor del contenido para describirlo (ej. `<h1>` para un encabezado).

* **Estructura:** Definen la jerarquía y organización de la página (ej. `<html>`, `<body>`, `<head>`).

* **Instrucciones:** Informan al navegador cómo interpretar y presentar el texto e imágenes.

<h2>Partes de un Tag:</h2>

* **Etiqueta de apertura:** `<nombre_etiqueta>` (ej. `<a>`).

* **Contenido:** El texto o elemento a marcar.

* **Etiqueta de cierre:** `</nombre_etiqueta>` (ej. `</a>`).

* **Autocierre:** Algunas etiquetas no necesitan cierre (ej. `<br>`, `<img src="foto.jpg">`).

<h2>Ejemplos Comunes::</h2>

* **`<h1>:`** Encabezado principal.

* **`<p>:`** Párrafo de texto.

* **`<a>:`** Enlace (hipervínculo).

* **`<img>:`** Imagen.

* **`<body>:`** Contenido visible de la página.

* **`<!DOCTYPE html>:`** Define la versión de HTML.

En resumen, los tags son los bloques de construcción esenciales que le dan sentido y forma visual a la web, permitiendo que navegadores como Chrome o Firefox entiendan y muestren correctamente el contenido.
